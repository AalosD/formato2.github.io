Proceso de diseño Web Mobile First

Etapa de diseño

Fase 1: Mapa del Sitio

Define la estructura de navegación de tu sitio web. Crea un mapa visual donde representes las páginas principales y sus conexiones.

Sube el mapa de sitio en formato .jpg, .png, o .gif y después contesta las preguntas para continuar a la siguiente etapa.



Fase 2: Wireframes

Crea wireframes para estructurar el contenido de cada página antes de realizar el diseño visual.

Wireframes de 480px


Wireframes de 768px


Wireframes de 1024px






Fase 3: Diseño Visual

Desarrolla el diseño visual de tu sitio web basándote en los wireframes y prototipos previos.




Fase 4: Creación de Contenidos

Crea y organiza el contenido de tu sitio web, como textos, imágenes, y otros elementos multimedia.




¡Has completado todas las fases!

Ahora tu proyecto web tiene una base sólida para continuar con el desarrollo.
